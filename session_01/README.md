# Session 1: Python, NumPy & PyTorch for NLP
Welcome to the first practical session of the NLP class!  
This session lays the foundation for the entire course. We’ll refresh essential Python skills, learn efficient numerical computing with NumPy, and get started with PyTorch — the deep learning framework we’ll use for all neural NLP models.
Here is the [link](https://docs.google.com/presentation/d/1S0XVSSNYr9lz1TRCMG49-stoOovmu8lK_nuPQ8ZipiU/edit?usp=sharing) to the presentation

## 🎯 Learning Objectives
By the end of this session, you will be able to:
- Write clean, efficient Python code for text processing
- Use NumPy arrays and operations for vectorized computations
- Understand PyTorch tensors and basic tensor operations
- Move between NumPy and PyTorch seamlessly (Have not covered during lecture but you may find the codes in notebook file)
- Implement simple NLP preprocessing steps (tokenization, vocabulary mapping, one-hot vectors) (Have not covered during lecture but you may find the codes in notebook file)

## 📦 Prerequisites
- Basic Python knowledge (loops, functions, lists, dictionaries, strings) would be nice
- Python 3.8+ installed on your machine
- A code editor (VS Code, PyCharm, or Jupyter Notebook)

## 🛠 Setup Instructions
Clone the repository and set up the environment:

```bash
git clone https://github.com/your-repo/nlp-class.git
cd nlp-class/session1
Download Miniconda (if not installed)-> https://docs.conda.io/en/latest/miniconda.html and follow the steps on this website.
conda create -n nlp_course python=3.13
conda activate nlp_course
conda install jupyter numpy pytorch
Either open your IDE (PyCharm, VS Code, Cursor etc..)
Create your first python/notebook file with `touch` command like the following
  touch nlp_session_01.py
  touch nlp_session_01.ipynb
```
